# mind-sharing-map
마음 나눌지도 (Mind Sharing Map)
프로젝트 소개

마음 나눌지도는 사용자가 기록한 감정과 위치 정보를 기반으로 감정 데이터를 시각화하는 위치 기반 감정 분석 서비스입니다.

사용자는 자신의 감정 상태와 위치를 기록할 수 있으며, 시스템은 해당 데이터를 분석하여 지도에 시각화합니다. 이를 통해 특정 장소와 감정의 관계를 확인할 수 있으며, 자신의 감정 흐름을 직관적으로 파악할 수 있습니다.

또한 커뮤니티 기능을 통해 사용자 간 감정 공유와 소통이 가능하도록 설계하였습니다.

문제 정의

현대인은 다양한 감정을 경험하지만 자신의 감정 상태를 체계적으로 기록하거나 객관적으로 확인하기 어렵습니다.

또한 특정 장소와 감정 상태 사이의 관계를 파악하기 어려워 자신의 정서적 패턴을 이해하는 데 한계가 있습니다.

본 프로젝트는 위치 정보와 감정 데이터를 결합하여 사용자의 감정 상태를 시각적으로 제공하고, 감정 데이터를 기반으로 정서적 이해를 돕는 것을 목표로 하였습니다.

문제점 및 해결 과정

프로젝트 진행 과정에서 '중립'과 '감정없음'의 구분이 어려운 문제가 발생하였습니다.

예를 들어,

"오늘은 평범한 하루였다."

와 같은 문장은 중립 감정으로 분류할 수도 있고 감정이 없는 진술로 볼 수도 있어 데이터 라벨링 과정에서 일관성을 유지하기 어려웠습니다.

또한 사람마다 동일한 표현에 대해 느끼는 감정과 강도가 달라 감정 분류 기준을 정의하는 과정에서도 어려움이 있었습니다.

이를 해결하기 위해 팀 내에서 감정 분류 기준을 정립하고 데이터 검수 과정을 반복하여 라벨의 일관성을 높이고자 노력하였습니다.

배운 점

본 프로젝트를 통해 데이터 수집, 전처리, 감정 레이블링, 모델 학습까지 AI 프로젝트의 전체 과정을 경험할 수 있었습니다.

특히 데이터 품질과 레이블링 기준이 모델 성능에 큰 영향을 미친다는 점을 체감할 수 있었으며, 협업과 의사소통의 중요성을 배울 수 있었습니다.

또한 실제 사용자 데이터를 다루는 과정에서 데이터 기반 문제 해결 능력을 향상시킬 수 있었습니다.
